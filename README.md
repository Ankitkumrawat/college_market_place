# CampusConnect 🎓

**CampusConnect** is a modern, premium, and fully-responsive web application designed for college campus ecosystems. It serves as a secure, peer-to-peer marketplace and community forum where verified students can trade items, participate in discussions, and get academic insights via an AI Smart Advisor.

---

## 🌟 Key Features

### 1. 🛍️ College Thrift Marketplace
- **Listing & Discovery:** Browse products listed by fellow students with clear categories, prices, condition tags, and seller details.
- **Smart Filtering:** Quick search bar and category-based navigation (Notes, Electronics, Books, Lab Equipment, Hostel Essentials, etc.).
- **Interactive Wishlist:** Bookmark items you are interested in for quick access.
- **Sell Form:** List new products with dynamic forms, condition descriptions, and upload specifications.

### 2. 💬 Community Discussion Forums
- **Peer Discussions:** Share questions, host discussions, or ask for recommendations.
- **Interactions:** Upvote posts that are helpful and leave nested comments under discussion threads.
- **Tags:** Filter posts by topics like Academics, Hostels, Events, or General.

### 3. 🤖 AI Smart Match Advisor
- **Personalized Recommendations:** Interact with a smart system that matches students with study resources, past exam notes, project partners, or specific thrift deals.
- **Predefined Queries:** Quick search templates for common campus scenarios.

### 4. 💬 Real-Time Chat System
- **In-App Messaging:** Buyers can initiate chats directly from product cards.
- **Interactive Chat Box:** Simple and elegant messaging layout to coordinate meeting places and negotiate deals on campus.

### 5. 🔐 Protected Auth Flow
- **Clean Standalone Login & Register:** Beautiful full-screen, focused card interface.
- **Secure Route Guards:** Restricts main dashboard access until the user successfully logs in.
- **Verification System:** Verification badge indicators for student accounts registered with `.edu` or `.ac.in` domain emails.
- **Fast Demo Presets:** Click-to-fill demo credentials for quick evaluation.

## 🚀 Getting Started

### 📋 Prerequisites
Ensure you have [Node.js](https://nodejs.org/) (version 16 or above) installed on your system.

### 💻 Local Setup

1. **Clone the Repository** (or unzip files to your directory):
   ```bash
   cd college_by
   ```

2. **Install Dependencies:**
   ```bash
   npm install
   ```

3. **Run the Development Server:**
   ```bash
   npm run dev
   ```
   The app will run locally on `http://localhost:3000` (or `http://localhost:5173` depending on ports).

4. **Build for Production:**
   ```bash
   npm run build
   ```
